Get started with the recruitment API

Integrating with our Recruitment API may be useful if you want to create adverts using your existing systems. It will allow you to publish to multiple sites, as well as Find an apprenticeship.

You can also use the Display adverts API to display adverts on any website.

What does the API do?

APIs (Application Programming Interfaces) let applications share information and data with each other.

If you connect using our Recruitment API, you can use your own systems to create adverts on Find an apprenticeship. 

You can also use our Display adverts API to display your adverts on any website.

How do I use the API?

You'll need someone with technical knowledge to connect your software program with the API, for example a software developer.

You can share this API page with the person who is going to connect your software program with the API. 

It can take a while to connect using an API, so leave enough time to integrate it.

How do I access the API?

Accessing the Recruitment API and Display vacancies API using your Apprenticeship service account

If you have owner permissions you can log in to your Apprenticeship service account to access the API keys. 

You will need to provide your developer with a key so they can use the API.

Your developer will be able to try out the API using the sandbox provided.

Accessing the Display vacancies API without an Apprenticeship service account

You need to register or sign in to access the API key.

You will need to provide your developer with a key so they can use the API.

Your developer will be able to try out the API using the sandbox provided.
