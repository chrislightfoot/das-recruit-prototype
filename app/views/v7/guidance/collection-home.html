Recruit an apprentice

How to use the Recruit an apprentice service to create, post and manage apprenticeship adverts.

  2. — Employers

Recruit an apprentice is the official government service for posting and managing apprenticeship adverts. Candidates can view and apply for apprenticeships on Find an apprenticeship.

Information for employers on how to create and manage apprenticeship adverts, along with guidance on using our recruitment APIs.

Our recruitment APIs may be useful if you want to create adverts using your existing systems or display adverts from Find an apprenticeship on your websites.

Display adverts API

You can use our Display adverts API to display adverts from Find an apprenticeship on any website, even if you don’t have an Apprenticeship service account.

APIs (Application Programming Interfaces) let applications share information and data with each other.

You'll need someone with technical knowledge to connect your system with the API, for example a software developer.

Legacy tools

The RSS feed and Av live legacy tools will no longer be available from March 2022 and our new Display adverts API should be used instead.
